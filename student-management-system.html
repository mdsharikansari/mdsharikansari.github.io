Project Description

This is a RESTful API for a simple Student Management System, built with Spring Boot. The primary focus of this project was to design and implement a Many-to-Many (@ManyToMany) database relationship. It allows you to create students and courses, and then enroll students into those courses.

Key Features

  • Designed a complex Many-to-Many database relationship between students and courses
  • Admin authentication and session management
  • CRUD for students and course assignments
  • Created relational endpoints to enroll students in specific courses
